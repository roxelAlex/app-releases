**Releases**.

## Установка (Android)

1. Скачайте `app.apk` по ссылке выше.
2. Откройте файл. При первой установке Android предложит разрешить установку из этого
   источника — подтвердите.
3. Готово.

## Проверка целостности (необязательно)

Рядом с APK публикуется файл контрольной суммы `app.apk.sha256`. Проверить:

```bash
sha256sum -c app.apk.sha256
```
